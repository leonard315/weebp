# **App Name**: CarStoreGo

## Core Features:

- Add to Cart: Allow users to add cars to a cart from the listing page.
- View Cart: Display the cart items on a dedicated cart page.
- Update Cart: Allow users to update the quantity of items in the cart and remove items.
- Price Calculation: Dynamically calculate and display the total price of items in the cart.
- Filtering: Enable make/model search and price range filtering on the car listing page.
- Car Detail Modal: Open a car detail modal when a user selects a car on the listing page. The modal should present complete and thorough details of a specific car.
- Inventory Management Tool: LLM Powered tool to offer suggestions and automations for how best to manage the inventory.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) for a trustworthy and professional feel.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Soft amber (#FFB300) for call-to-action buttons and highlights.
- Body font: 'PT Sans', a sans-serif font providing a balance of modernity and warmth suitable for both headings and body text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a set of consistent and professional icons to represent car features and functionalities.
- Implement a grid-based layout with clear visual hierarchy for car listings and cart items.
